# Fab.Internet

`Fab.Internet` is an Elixir library for generating random internet related data.

## Installation

Add `:fab_internet` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:fab_internet, "~> 1.0"}
  ]
end
```
